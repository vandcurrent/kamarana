Kamarana

Kamarana.com merupakan sebuah sistem yang digunakan untuk berbagi informasi dan pengalaman serta panduan kepada wisatawan mengenai semua fasilitas yang ada di suatu daerah berasarkan pengalaman yang sudah dilaluinya. Kelebihan dari aplikasi ini adalah setiap anggota dapat mencari dan berbagi informasi segala perjalanan yang pernah dikunjungi. Hal yang paling penting adalah informasi mengenai transportasi, penginapan, makanan dan budget segala sesuatunya sudah disediakan, sehingga user yang memiliki pengalaman dapat membuat cerita tersediri tentang pengalaman berwisatanya.

This theme is a fixed layout with two columns. All of the information within the main content area is nested within a div with an id of "primaryContent". The sidebar's (column #2) content is within a div with an id of "secondaryContent".

Homepage

Pada Homepage atau halaman depan, user dapat melihat tulisan-tulisan cerita perjalanan orang lain dengan mengklik Jelajah. Untuk memulai membuat tulisan, user harus Login terlebih dahulu, atau Daftar Jika belum memiliki akun Kamarana.

How To Edit Color

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

#primaryContent a {
  color: #someColor;
}

Change Structure

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

#wrap #primaryContent a {
  color: #someColor;
}

Styles Styles customization and highlights

Heads up! I'm using two CSS files in this theme. The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

Customize Slider

If you want to change the slider's speed transition, open up the file scripts.js and change the following code: 

/* ---------- @ SlidesJS -----------*/
$('#banner').slides({
  preload: true,
  generateNextPrev: true,
  autoHeight: true,
  effect: "slide",
  play: 5000
});
